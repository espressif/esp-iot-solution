# Changelog

## v0.1.0

### Features
- Initial release of Lottie player for LVGL
- Support for LVGL v8 and v9
- ThorVG-based rendering engine
- Playback control: play, pause, stop, restart
- Loop control with configurable count
- Segment playback support
- Custom frame delay support
- Multiple buffer modes: auto-allocated, user-provided, or draw buffer
