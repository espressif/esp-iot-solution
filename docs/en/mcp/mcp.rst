MCP Service
==============================
:link_to_translation:`en:[English]`

==================  ===========  ===============  ===============  =============
Supported chips     ESP32        ESP32-C2         ESP32-C3         ESP32-S3
==================  ===========  ===============  ===============  =============

MCP Service provides a simplified API interface for accessing common MCP functions. It supports HTTP/SSE/MQTT/WS and other common scenarios.
